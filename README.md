# TS-Express-Starter

## Description

Basic Express API with a single `Hello World!` endpoint.

## Requirements

- Node version 22.8.0
- pnpm version 9.12.2
- Docker

## Getting started

### Install dependencies

```bash
pnpm install
```

### Run local server with hotreload

```bash
pnpm dev
```

### Run with Docker

```bash
pnpm docker:build
pnpm docker:run
```
